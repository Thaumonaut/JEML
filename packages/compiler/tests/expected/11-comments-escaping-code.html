Syntax reference

This document shows how to escape special characters like *asterisks*, _underscores_, and &ampersands&. Literal backticks need ` escaping too.

Code example

Here is an inline code reference: const x = 1.

function greet(name: string): string {
  return `Hello, ${name}!`;
}

Another language

def greet(name):
    return f"Hello, {name}!"
